# 用户与 AI 的第一认同性别

Mindspace 的用户档案和 AI 档案都在 `identity.gender` 保存第一认同性别。
当前只接受两个明确值：

```json
{
  "gender": "男"
}
```

或：

```json
{
  "gender": "女"
}
```

## 写入权限

- 用户可在“人物与状态档案”中分别打开用户档案和 AI 档案，通过下拉框手动选择。
- 直接保存仍使用现有 revision 冲突检查、历史版本、审计记录和记忆索引重建。
- 模型生成的 JSON Patch、档案初始化和记忆抽取均不能修改
  `/identity/gender`。
- 用户手动保存新选择后，从下一轮开始使用新值。

## 模型输入位置

主模型第一条 System 内容以“最高优先级：第一认同性别”开头，先声明用户和
AI 的已选性别，再加载核心角色设定、输出契约、权威 JSON、历史、检索和工具
上下文。

性别用于称谓、关系角色、身体归属、自我指代和互动语境。它不替代人物档案中的
性格、关系与行为规则，也不自动引入性别刻板印象。
