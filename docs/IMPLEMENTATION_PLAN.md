## 设计原则

- 默认可运行：没有模型和音频服务也能启动并完成闭环。
- 模型只产出候选：所有写回都必须经过程序策略。
- 停止优先：取消后允许完成已发出的阻塞模型调用，但禁止进入持久化。
- 服务可替换：模型、检索、档案、会话、审计、TTS、ASR 都由适配器隔离。
- 源项目只读：迁移前使用独立 runtime，不复用原项目的可写路径。
