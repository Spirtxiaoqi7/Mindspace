# 更新历史

Mindspace 使用语义化版本号。机器可读的同源记录位于
[`docs/release-history.json`](docs/release-history.json)。

## 0.5.9 - 2026-07-24

### 用户角色卡与第一认同性别

- 支持将本地人物自述压缩为结构化用户档案，通过现有档案 API 写入，
  保留历史版本、revision、审计记录和记忆索引；私人档案不进入源码或安装包。
- 用户档案与 AI 档案的 `identity.gender` 只接受“男”或“女”，人物档案
  表单提供独立的手动选择器。
- 性别属于用户所有的只读模型身份字段；模型 JSON Patch、档案初始化和记忆
  抽取不能自行修改。
- 主模型第一条 System 内容先声明用户与 AI 的第一认同性别，再进入角色设定、
  权威 JSON、历史、检索和工具上下文。
- 性别用于称谓、关系角色、身体归属和自我指代，但不替代人物档案中的性格、
  关系与行为规则。

## 0.5.8 - 2026-07-24

### 通话与面对面互动

- 点击实时语音时先选择“通话”或“面对面”；默认仍为原有通话逻辑。
- 面对面模式提供可选场景输入框，并持久保存上次模式和场景内容。
- 每轮语音请求携带受校验的 `voice_context`，面对面场景以临时高优先级
  System 层进入模型输入。
- 角色可自然描述自身外观、动作、距离和体感线索，但不能替用户断言动作、
  反应或感受。
- 场景属于用户控制的互动配置，不进入长期对话历史、人物事实、向量记忆或
  JSON Patch 证据。

## 0.5.7 - 2026-07-23

### 可信输入、运行恢复与调用预算

- 模型调用按规划、研究复核、正文、协议修复和记忆抽取独立限额，单轮总上限为五次。
- 普通闲聊不会因“今天”等时间词误触联网；已有可用正文时由服务端确定性补齐安全协议。
- 流式正文按检查点持久化，刷新或断线只恢复订阅，Core 重启保留已生成部分且不自动重跑。
- 检索、工具和调度信息降为审计可见，ASR 低置信候选只在当前轮临时参考。
- 用户可从人物卡直接编辑档案、检查 revision 冲突并恢复历史版本。
- 新增默认脱敏的模型输入检查器。
- 中文 ASR 增加回声排除、打断限频、重复抑制和低置信草稿仲裁。
- 外部能力严格串行执行，HTTP 客户端保持连接池复用。
- Launcher 在创建窗口前初始化更新与运行时管理器，消除首屏状态查询竞态。

## 0.5.6 - 2026-07-23

### 档案写回与时间、联网判断修复

- 人物与状态档案新增表单编辑、保存状态和明确错误反馈，同时保留高级 JSON 模式。
- 有记忆价值的语句才触发结构化抽取，AI 自述仅凭本轮逐字证据写入 agent 字段。
- 明确联网请求在私有规划失败时保留确定性调用，并拦截未执行却声称已搜索的表述。
- 服务端直接给出日期、星期、周末与明日判断，避免模型自行换算出错。
- 简短确认不再自动等同于结束话题、解决情绪或准备休息。

## 0.5.5 - 2026-07-23

### 中文整句复核与 CUDA 调度

- 保留 Paraformer 实时字幕，新增 Fun-ASR Nano 中文整句复核与无模型安全回退。
- 含糊尾词、句末和播放场景使用动态断句，旧默认 250 ms 迁移为 600 ms。
- CUDA 推理单通道且流式优先，Nano 固定单批次、中文和有限热词以压低占用。
- Launcher 等待 ASR 加载完成后再启动本地 TTS，避免大模型并行搬运和显存争抢。
- 最终事件保留流式初稿、复核原稿、纠偏结果、端点原因和明确回退状态。

## 0.5.4 - 2026-07-23

### 低延迟与断线恢复

- 本机状态改为能力触发时采集，普通对话不再无条件扫描本机。
- 向量查询单飞并复用既有向量，避免同轮分支和二次工具调用重复编码。
- HTTP 客户端统一连接池复用。
- 流式事件支持序号、重连续传与重复过滤，前端按动画帧批量渲染。
- 启动器合并并发启动、并行健康探测并防止停止期间误拉起。
- 情绪模型和运行链路移除，仅保留明确的零开销接口边界。

## 0.5.3 - 2026-07-22

### 角色优先与事实约束

- 角色设定提升为首条 System，回复首先从人物关系、性格和当下感受出发。
- AI 只表示存在媒介，不再触发通用助手、客服或问答口吻。
- 召回内容改为候选线索，未获当前输入、权威 JSON 或近期原始对话确认时不得引用为个人事实。
- 空字段明确表示未知，不再由常识、相似语义或知识库补成人物偏好。
- JSON 输出协议压缩并后置，与用户可见的角色回复分离。

## 0.5.2 - 2026-07-22

### 实时识别与智能词表

- 新增自适应环境噪声基线，呼吸和瞬时噪声不再直接打断朗读。
- 语音片段先合并再发送，生成期间的补充话会与原输入组成同一轮重发。
- 播放中未达到打断条件的有效语音延后保留，不再丢失或拆成多轮。
- 新增可在线编辑的识别词表，人物名称和专用名词高强度增强。
- 人物与状态 JSON 自动投影为识别热词，不进入 Prompt，也不增加 LLM 判断负担。
- 识别纠偏元数据独立审计并限制容量增长。

## 0.5.1 - 2026-07-21

### 时间感知与自然续接

- 文字与语音统一记录服务端 UTC 时间、当地时区和对话间隔。
- 新增可配置的沉默后自然续接，文字和语音可分别设置等待时间。
- 主动续接不伪造用户输入、不催促用户回应，也不会触发 JSON 写入。
- 语音插话记录实际播放位置，向下一轮区分已听和未听内容。
- 播放中先降低音量，VAD 或 ASR 确认人声后才真正打断。

## 0.5.0 - 2026-07-21

### 启动器分类与可靠下载

- 首页改为状态概览与基础环境、语音能力、下载存储、更新维护四个分类。
- 基础环境按真实依赖顺序安装，自动复用已就绪组件和断点缓存。
- 下载失败显示错误码、阶段、操作编号，并可导出脱敏诊断报告。
- 更新应用时保留私有环境、模型、数据和下载缓存。
- 增加 GPT-SoVITS V4 角色音色启动、停顿和韵律修复。

## 0.4.7 - 2026-07-20

- 新增历史版本公告记录。

## 0.4.6 - 2026-07-20

- 恢复 ASR 与 CosyVoice 共享运行时。
- 普通更新优先使用 Core 原子替换。

- 修复更新时错误清理私有环境的问题。
